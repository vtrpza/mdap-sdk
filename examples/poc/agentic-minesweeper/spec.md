# Minesweeper Specification

Build a complete Minesweeper game with the following requirements:

## Game Rules

- 16x16 grid
- 40 mines randomly placed
- Left-click to reveal a cell
- Right-click to flag/unflag a cell
- Revealing a mine = game over (lose)
- Revealing all non-mine cells = win
- Numbers show count of adjacent mines (1-8)
- Empty cells (0 adjacent mines) auto-reveal neighbors

## Technical Requirements

- Single HTML file with embedded CSS and JavaScript
- Vanilla HTML/CSS/JS (no frameworks)
- Must work in modern browsers

## UI/UX Requirements

- Clean, modern design
- Visually distinct states: hidden, revealed, flagged, mine
- Clear visual feedback on hover
- Win/lose state clearly displayed
- New game button
- Mine counter (remaining mines = total - flags)
- Timer showing elapsed seconds
- Responsive grid that looks good

## Quality

- No console errors
- Smooth interactions
- Polished feel
